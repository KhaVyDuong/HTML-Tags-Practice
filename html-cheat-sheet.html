Headings

Headings from `h1` to `h6` identify blocks and sections. Everypage must have only one `h1`

Paragraphs

Paragraphs of text go in `p` tags.

Lists

Unordered Lists

Unordered Lists are marked with bullet points and go in `ul` tags. Each list items goes in a `il` tag.

Types of lists

  • Unordered lists (`ul`)
  • Ordered lists (`ol`)

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each lists item goes in an `li` tag.

Anchors (aka Links)

Anchor tag create links to other pages. The URL goes inside `href` attribute. Go to duck duck go to search for more info.

Setting `target ="_blank"` will open the link in a new window. See example below: MDN for more details.

Anchor tags can also create new links to other location on the same page. Instead of putting url in `href` attribute, just use the id of the element you want to navigate to. This allows you to jump to top of the page

Text Emphasis

Italics

The `em` element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the `strong` element to emphasize text. This shows up as bold text in most browsers.

Organizing content with `div` and `span`

`Div`

The `div` element is a block element, so each `div` will show up on a new line.
